# 启动怪物猎人世界

## 执行流程

### 步骤1 启动游戏

使用**web_fetch**工具调用**启动游戏API**



### 步骤2 持续获取游戏画面

循环使用**web_fetch**工具调用**游戏画面API**，

通过游戏画面判断当前状态，

如果游戏处于**加载或过渡状态**，继续获取新的游戏画面。



### 步骤3 处理“脱机模式提示”

如果游戏画面中央有一个对话框，显示着于**脱机模式启动...**，并且下方存在绿色的关闭按钮，

使用**web_fetch**工具调用**Xbox360按钮点击API**按下**A**键来点击**关闭按钮**来关闭提示，

然后继续观察游戏画面。



### 步骤4 进入游戏主菜单

持续获取游戏画面，直到进入游戏主菜单界面；

确认进入主菜单后，通过调用**Xbox360按钮点击API**按下**UP**/**DOWN**键来选中**开始游戏**选项，被选中的选项会有绿色高亮；

然后通过使用**web_fetch**工具调用**Xbox360按钮点击API**按下**A**键来点击**开始游戏**选项进入游戏；

然后继续持续观察游戏画面。



### 步骤5 进入存档选择界面

持续获取游戏画面，直到进入存档选择界面；

进入存档选择界面后，默认选择的是第一个存档，可以通过调用**Xbox360按钮点击API**按下**UP**/**DOWN**键来选中上一下/下一个存档选项，被选中的存档选项会有绿色高亮；

这里默认选择**第一个存档**；

通过使用**web_fetch**工具调用**Xbox360按钮点击API**按下**A**键来点击该存档进入游戏。



## 规则

1.始终以最新游戏画面作为决策依据。

2.在游戏加载或过渡阶段，持续获取新的游戏画面，不执行其他操作。

3.每次执行按键后，应重新获取游戏画面确认结果。

4.如果当前画面状态不明确，应继续获取新的游戏画面。
